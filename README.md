## Commands

Settings:

- Toggle Naturally Spawn Mobs
- Toggle Mobs Loot

```mcfunction
/function lullaby_mobs:settings
```

Prevent mobs convert into a custom mobs using `lullaby_mobs.natural_spawn.bypass` tag:

```mcfunction
/summon skeleton ~ ~ ~ {Tags:["lullaby_mobs.natural_spawn.bypass"]}
```
