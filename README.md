## Commands

Settings:

- Toggle Naturally Spawn Mobs
- Toggle Mobs Loot

```mcfunction
/function lullaby_mobs:settings
```

Disable:

```mcfunction
/datapack disable "file/lullaby-mobs.zip"
```

Enable:

```mcfunction
/datapack enable "file/lullaby-mobs.zip"
```
